## ✨ Key Innovations Documented

1. **Three-Layer Scope Isolation** [NOVEL]
   - Layer 1: Gateway capabilities
   - Layer 2: Bilateral negotiation
   - Layer 3: Doorman runtime enforcement

2. **Doorman Access Check** [NOVEL]
   - 6-step validation process
   - O(1) peer lookup, O(k) intent matching
   - Combines cryptographic identity + scope grants

3. **Symmetric Scope Mirroring** [NOVEL]
   - UX default: auto-grant peer's offered scopes
   - Bilateral trust with config on ONE side only
   - Asymmetric override via flags

4. **Hierarchical Topic Policies** [NOVEL]
   - 4-level fallthrough
   - Fine-grained agent control without complex ACLs

5. **Stable Cryptographic Identity**
   - Ed25519 public key-based (not URL-based)
   - Survives tunnel rotation, DHCP changes

6. **Sliding Window Rate Limiting**
   - Precise retry-after calculation
   - Per-peer, per-intent tracking
