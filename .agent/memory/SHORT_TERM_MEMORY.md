> Active context for the current task or session. Ephemeral. Cleared or rotated when a task closes. This is your whiteboard, not your archive.

**Not for:** long-term facts (use `../MEMORY.md`), personality (use `../SOUL.md`), bead state (use `BEADS.md`).
**Is for:** the specific intermediate state of what you're working on right now.

## Hydrating from long-term memory

Keep this file small. When you need more, pull from the right place:

- **Facts about this repo** → `../MEMORY.md`
- **Patterns + rules** → `../PATTERNS_CATALOG.md` (130 patterns, ID-indexed) or `../KNOWLEDGE_PACK.md` (article index)
- **Personality / behavior** → `../SOUL.md`
- **Operating rules** → `../AGENT.md`
- **Past mistakes** → `../LESSONS_LEARNED.md`
- **Anything from source articles** → `../skills/search-substack.sh "topic"` returns title + URL + attribution

Rule: if you think you're remembering something that isn't here, it's in one of the above. Look before asking the human.

- If the human says "fresh context," archive + reset immediately.
- Keep this file under ~2KB. If it grows, summarize and archive the rest.
